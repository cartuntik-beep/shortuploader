Privacy Policy

Stand:

1. Überblick

Diese Datenschutzerklärung beschreibt, wie ShortUploader mit Daten umgeht. ShortUploader ist eine Desktop-Anwendung, die lokal auf deinem Gerät ausgeführt wird. Diese Website dient primär der Dokumentation und Bereitstellung rechtlicher Informationen.

2. Datenverarbeitung in der Anwendung

  • Keine Server-Speicherung: ShortUploader speichert Videos nicht auf einem externen Server.
  • Lokale Verarbeitung: Metadaten (z. B. Titel, Beschreibung) werden innerhalb der App lokal verarbeitet.
  • Keine Datenweitergabe: Es erfolgt keine Weitergabe von Nutzerdaten an Dritte durch die App (außer an die Zielplattform im Rahmen des Uploads).

3. Uploads zu Plattformen

Beim Upload werden Inhalte und ggf. Metadaten an die von dir ausgewählte Plattform übertragen. Für die Verarbeitung dort gelten die Datenschutzbestimmungen und Bedingungen der jeweiligen Plattform.

4. Website (GitHub Pages)

Diese Website wird über GitHub Pages bereitgestellt. GitHub kann dabei technische Zugriffsdaten (z. B. IP-Adresse, User-Agent, Zeitpunkt) verarbeiten, um die Website auszuliefern und zu schützen. Details findest du in den Datenschutzinformationen von GitHub.

5. Cookies & Tracking

Auf dieser Website werden von uns keine eigenen Tracking-Tools eingesetzt. Es werden keine Werbe-Tracker eingebunden.

6. Änderungen

Diese Datenschutzerklärung kann aktualisiert werden. Die jeweils aktuelle Version ist auf dieser Seite veröffentlicht.

7. Projekt / Repository

Offizielles Projekt-Repository: github.com/cartuntik-beep/shortuploader

Hinweis: ShortUploader ist nicht mit TikTok, YouTube oder anderen Plattformen verbunden und wird nicht von ihnen unterstützt.
