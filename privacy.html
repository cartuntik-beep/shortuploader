Privacy Policy

Last updated:

1. Overview

ShortUploader is a locally installed desktop application. The application does not operate its own servers and does not collect personal data for its own purposes.

2. Data Collection

The application does not:

  • Collect analytics or tracking data
  • Store personal data externally
  • Share user data with third parties

3. Authentication

When connecting a platform account, authentication is performed through official authorization systems provided by the platform.

Only the permissions explicitly approved by the user are used.

4. Uploaded Content

Videos and metadata are transmitted directly from the user's device to the selected platform.

ShortUploader does not store or analyze uploaded content.

5. Website Hosting

This website is hosted via GitHub Pages. GitHub may process technical access data in accordance with their own privacy policy.

6. Cookies

This website does not use its own cookies or tracking tools.

7. Affiliation Disclaimer

ShortUploader is not affiliated with or endorsed by TikTok, ByteDance or any other platform provider.
