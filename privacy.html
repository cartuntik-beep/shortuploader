Privacy Policy

Effective date:
This Privacy Policy explains how ShortUploader (“the Application”) handles data.

1. Data Minimization

ShortUploader respects user privacy and processes only the minimum data required for application functionality.

2. Authentication Data

The Application may store authentication data such as access tokens and refresh tokens locally on the user’s device to enable publishing features and maintain session state.

3. No Selling or Advertising Use

  • No personal data is sold or rented.
  • No personal data is used for advertising purposes.
  • The Application does not operate external tracking systems.

4. Data Sharing

ShortUploader does not transmit user data to third-party servers outside of the selected publishing platforms. Network requests are limited to the platform APIs required for authentication and posting features.

5. User Control and Deletion

All stored data remains under the user’s control and can be removed at any time by logging out or deleting local application data.

6. Contact

For questions, please use the GitHub repository: cartuntik-beep/shortupload.
